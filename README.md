# FoggyWorld
A computer graphics project about a foggy world!

You can explore more here: https://1160275520.github.io/FoggyWorld/

Key Press Instruction:
Go close/far away: W/S
Go left/right:A/D
Go up/down: Q/E
Change RGB color: O/P, U/I, K/L
